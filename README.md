# 📊 CodeAlpha - Exploratory Data Analysis (EDA) on Netflix Dataset

## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**. The objective is to perform **Exploratory Data Analysis (EDA)** on the Netflix Titles dataset to understand the data, identify trends, detect missing values, and visualize meaningful insights.

---

## 🎯 Objectives

- Load and explore the Netflix dataset
- Understand the dataset structure
- Identify and handle missing values
- Check for duplicate records
- Generate descriptive statistics
- Create meaningful visualizations
- Draw insights from the data

---

## 🛠️ Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset

**Dataset:** Netflix Titles Dataset

The dataset contains information about Netflix Movies and TV Shows, including:

- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

---

## 📈 Exploratory Data Analysis Performed

- Displayed first and last five rows
- Checked dataset shape
- Viewed column names
- Explored dataset information
- Analyzed missing values
- Checked duplicate records
- Generated statistical summary
- Created data visualizations
- Summarized key findings

---

## 📊 Visualizations

- Movies vs TV Shows (Bar Chart)
- Top 10 Countries with Netflix Content
- Most Common Content Ratings
- Distribution of Release Years
- Movies vs TV Shows (Pie Chart)

---

## 🔍 Key Insights

- The dataset contains **7,787 Netflix titles** and **12 columns**.
- Movies outnumber TV Shows on Netflix.
- Most Netflix content was released after **2010**.
- The United States contributes the highest number of Netflix titles.
- Some columns contained missing values, which were identified during data cleaning.
- Netflix provides content suitable for different age groups through a variety of content ratings.

---

## ✅ Conclusion

This project demonstrates the process of Exploratory Data Analysis (EDA) using Python. By cleaning, exploring, and visualizing the Netflix dataset, meaningful insights were obtained about content distribution, release trends, and ratings. The project highlights how EDA supports data-driven decision-making and serves as a strong foundation for further analysis.
